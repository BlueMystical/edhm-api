Elite Dangerous HUD Mod (EDHM)

Ist ein Mod, erstellt von CMR GeorjCostanza @psychicEgg
EDHM ermöglicht präzise Farbänderungen an fast jedem Element auf dem HUD ohne die lästigen Nebenwirkungen von die XML-Methode.
Der Mod ist eine Anwendung der Shader-Modding-Software 3Dmigoto und daher nur auf dem PC verfügbar.

  • Ursprünglich veröffentlicht bei FDev Foren.
  • Und beim Autor GitHub-Seite
  • hier ist 3Dmigoto

Es funktioniert auch, um die Kabinenbeleuchtung in eine beliebige Farbe oder sogar zweifarbig zu ändern, wenn Sie dies bevorzugen.
Wir haben auch einige der Armaturenbrettleuchten, die direkt in Ihr Gesicht scheinen, gedimmt, um die Ermüdung der Augen zu verringern.

EDHM - Benutzeroberfläche

EDHM_UI ist die Benutzeroberfläche für EDHM, entwickelt von @Blue Mystic.

Um EDHM allein zu verwenden, müssen Sie komplexe Konfigurationsdateien manuell bearbeiten.
Das ist für den durchschnittlichen Benutzer nicht geeignet, daher hatte 'Blue Mystic' mit Erlaubnis des Autors von EDHM ein Benutzeroberflächenprogramm für EDHM erstellt.

Zu den aktuellen Funktionen gehören:

  • Ermöglicht dem Benutzer, alle Optionen über eine einfache grafische Oberfläche zu ändern.
  • Mit der neuesten EDHM-Version gebündelt – EDHM muss nicht separat installiert werden.
  • Gebündelt mit vielen vorgefertigten, von der Community bereitgestellten Designs.
  • Voll kompatibel mit Horizons und Odyssey!
  • Verfügbar als Open Source auf GitHub Seite

Laden Sie das EDHM-UI-Setup-Programm herunter

EDHM_UI wird mit der neuesten verfügbaren EDHM-Version geliefert (bedeutet, dass Sie Sie müssen NUR die Benutzeroberfläche herunterladen, um BEIDES zu haben) und viele Farbdesigns, die von der Community erstellt wurden.

Sie können das neueste Installationsprogramm für EDHM-UI herunterladen von:

  • Der GitHub Seite
  • Dieser Direktlink

Schließen Sie das Spiel, bevor Sie mit der Installation beginnen

Regel Nr. 1 zum Modden von Spielen:

Schließe das Spiel, bevor du Mods installierst oder aktualisierst.

Das liegt daran, dass der Mod bestimmte Dateien berühren muss, die ausschließlich vom Spiel verwendet werden.

Wenn das Spiel läuft, wird die Mod nicht richtig installiert, also... Spiel schließen.

Erster Lauf

Beim ersten Start der Benutzeroberfläche werden Sie aufgefordert, den Pfad zu angeben die Spielordner.

Sie können die Pfade auf zwei Arten festlegen:

  • Manuell (für fortgeschrittene Benutzer) entweder durch Kopieren/Einfügen des vollständigen Pfads in das Textfeld oder Klicken Sie auf die Schaltflächen [...], um die ausführbare Datei des Spiels zu finden.
  • Automatisch, indem Sie auf die grüne Schaltfläche „Assistent“ klicken und den Anweisungen folgen.

Wird Ihnen eine Menge Kopfschmerzen ersparen, wenn Sie einfach auf den grünen Knopf klicken und tun, was er sagt.

Anleitung

Farbthemen verwenden

  • Design auswählen
  • Designs verwalten
  • Vorschau eines Designs anzeigen

Wählen Sie ein Thema aus

Am einfachsten beginnen Sie mit der Auswahl eines der vielen benutzerdefinierten Farbdesigns, die von der EDHM-Community erstellt und geteilt werden.

Gehen Sie dazu wie folgt vor:

  • Klicken Sie auf ein beliebiges Thema in der Liste.
  • Klicken Sie auf die Schaltfläche „Design anwenden“.
  • Alt + Tab zum Spiel und drücken Sie die Taste F11.

Der [F11] ist ein von EDHM installierter Bindungsschlüssel, der verwendet wird, um alle von Ihnen vorgenommenen Änderungen zu aktualisieren.

Designs verwalten

EDMH enthält viele Themen, und vielleicht sind nicht alle nach Ihrem Geschmack, das ist in Ordnung.

Sie können jedes Thema als 'Favorit' festlegen und durch Umschalten einer Schaltfläche können Sie filtern, um nur Ihre Favoriten anzuzeigen.

Sie können auch Designs importieren und exportieren, aber bei jedem Update des UI-Programms wird jedes neu freigegebene Design enthalten sein, sodass Sie nichts importieren müssen.

Wir würden etwas später über das 'Exportieren' von Designs sprechen.

Designvorschau

Alle Themen haben ein 'Miniaturbild', das in der Themenliste angezeigt wird und Ihnen eine Vorstellung davon vermitteln soll, wie ein Thema aussieht.

Aber zusätzlich haben alle Themen ein 'Big Preview'-Bild, das Sie sehen können durch:

  • Klicken Sie mit der rechten Maustaste auf ein beliebiges Thema in der Liste.
  • Klicken Sie auf das Menü "Designvorschau".
  • Das Bild wird in Ihrem Standard-Webbrowser angezeigt.

HUD-Elemente anpassen

Kennen Sie die Mod schon?

Wenn ja, dann können Sie Ihre eigenen Änderungen an den Hud-Elementen des Spiels vornehmen.

Das UI-Programm gruppiert die vielen verschiedenen modifizierten Elemente in Kacheln, die auf der linken Seite des Fensters angezeigt werden

Sie können auf eine dieser Kacheln klicken und alle Einstellungen werden auf der rechten Seite, dem Detailbereich, angezeigt.

Detailbereich

Es gibt verschiedene Arten von Einstellungen, die Sie anpassen können, damit das Spiel genau so aussieht, wie Sie es möchten.

  • Combo-Elemente: Sie können eine von wenigen Optionen auswählen.
  • Farbelemente: Sie können JEDE Farbe auswählen, die Sie mögen.
  • Schieberegler-Elemente: Wird verwendet, um die Helligkeit oder Transparenz vieler Elemente zu steuern.

Klicken Sie auf die linke Seite eines beliebigen Elements, um es auszuwählen, seine Beschreibung wird unten angezeigt.

Fahren Sie mit der Maus über ein beliebiges Element, um seine Beschreibung anzuzeigen und manchmal eine Vorschau des Elements anzuzeigen.

Farbwähler

Das UI-Programm verfügt über eine erweiterte Farbauswahl, mit der Sie jede gewünschte Farbe auswählen können.

Standardmäßig hat es 2 Farbpaletten:

  • Personalisiert: Zeigt alle im aktuellen Design verwendeten Farben an.
  • Standard: Zeigt eine Reihe von Standardfarben zur Auswahl an.

Klicken Sie auf die Schaltfläche Mehr Farben, um die erweiterte RGB-Farbauswahl anzuzeigen.

Wenden Sie Ihre Änderungen an

Wenn Sie Ihre Änderungen im Spiel sehen möchten, müssen Sie sie zuerst anwenden.

Klicken Sie auf die Schaltfläche Design anwenden, dann ALT+TAB zum Spiel und drücken Sie die Taste F11-Taste

Es ist sehr wichtig, dass Sie beim Vornehmen von Änderungen NICHT auf ein Thema in der Liste klicken, nicht einmal auf das Thema, das Sie gerade bearbeiten. Wenn Sie dies tun, werden die gespeicherten Einstellungen der angeklickten Design wird geladen und alle nicht gespeicherten Änderungen gehen verloren.

Wenn Sie die Änderungen anwenden, werden sie in den Aktuellen Einstellungen gespeichert, und die aktuellen Einstellungen werden jedes Mal überschrieben, wenn Sie ein Design anwenden.

Elemente suchen

In der Hauptleiste sehen Sie ein Suchfeld, mit dem Sie nach beliebigen Elementen suchen können.

Die Suche akzeptiert Teilnamen, sodass Sie jedes gewünschte Element leicht finden können.

Kann auch nach Themennamen und Themenautoren suchen.

Anleitung

Erstellen Sie Ihr eigenes Design

  • Wählen Sie ein Basisthema aus
  • Design speichern
  • Geben Sie ihm ein Miniaturbild
  • Design exportieren

Wählen Sie ein Basisthema aus

Sie können jedes Design als Basis für Ihr eigenes verwenden, wählen Sie es einfach in der Liste aus und beginnen Sie mit den Änderungen.

Es gibt 2 spezielle Themen, die Ihnen dabei helfen:

  • Das Elite Default – ORANGE-Theme, das, wie Sie sich vorstellen können, alle Einstellungen auf Standardwerte hat, also das Äquivalent zu einem nicht modifizierten Spiel ist.
  • Beim Design @XML – Farbimport sind alle Einstellungen auf die Verwendung von XML eingestellt. Verwenden Sie dies, wenn Sie alte XML-Farbkonfigurationen importieren möchten.

Wählen Sie ein Design aus und wenden Sie es an, dann können Sie auf die Aktuellen Einstellungen klicken und diese verwenden und Ihre eigenen Änderungen vornehmen.

Design speichern

Es ist immer eine gute Idee, Ihre aktuellen Einstellungen in ein Design umzuwandeln, um zu vermeiden, dass Ihre Daten bei einem neuen FDev-Update verloren gehen. Sie löschen gerne von Zeit zu Zeit alles im Spielordner. Designs werden in Ihrem Ordner Eigene Dateien gespeichert, wo niemand außer Ihnen sie berühren wird.

Wenn Sie also Ihre Einstellungen bereits so haben, wie Sie es möchten, ist es an der Zeit, sie als Design zu speichern. Klicken Sie auf die Schaltfläche Neues Design erstellen, geben Sie ihm einen Namen und legen Sie Ihren Autorennamen fest.

ACHTUNG: Wenn Sie den Namen eines bestehenden Designs verwenden, wird dieses Design überschrieben.

Ihr kürzlich erstelltes Design ist das einzige in der Liste ohne Miniaturbild.


Tool zum Erstellen von Miniaturansichten

Klicken Sie mit der rechten Maustaste auf Ihr Design und wählen Sie Miniaturansicht erstellen. Ein neues Fenster wird angezeigt.

  • Kopieren Sie einen Screenshot Ihres Spiels
  • Fügen Sie den Screenshot in das Thumbnail Maker Tool.
  • Ändern Sie die Größe des Fensters zum Vergrößern/Verkleinern
  • Bewege den rechteckigen Delimitador so, dass er dein HUD am besten abdeckt.
  • Klicken Sie auf die Schaltfläche Verarbeiten und speichern.

Das Bild wird PREVIEW.jpg heißen, Sie können diesen Namen NICHT ändern.

Das Bild wird in dem Ordner gespeichert, in dem sich Ihr Design befindet.

Design exportieren

Sie können jedes ausgewählte Design exportieren, um es entweder zu sichern oder mit der Community zu teilen.

  • Wählen Sie das Thema in der Liste aus.
  • Klicken Sie auf die Schaltfläche Design exportieren

Sie werden gefragt, wo Sie es speichern möchten, und Sie werden mit einer ZIP-Datei enden, die Ihr exportiertes Design enthält.

Wenn du es teilen möchtest, poste es auf unserem Discord Server im #Sharing-Colors Channel.

Sie können dort die ZIP-Datei und einen Screenshot Ihrer Farben in voller Größe hochladen.
